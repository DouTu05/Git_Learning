# 新建版本库

新建一个版本库，对本地的代码进行管理。版本库又叫**仓库**(Repository, Repo)。仓库类似一个目录，这个目录里面的所有文件都被Git管理起来，每个文件的删除、添加、修改都会被Git记录下来。以便任何一个时候都可以查看修改历史和还原到某一个版本。

创建一个仓库只需要将一个目录便为git可以管理的仓库就可以了。一般来说可以通过两种方式创建仓库：

* 方式1：git init (在电脑本地直接创建一个仓库)
进入该目录文件夹后输入下面命令，就能创建一个仓库了。
``` bash
git init <project-name>
```
创建成功后，目录文件夹下会出现一个隐藏的.git文件夹，该文件夹是git仓库的配置文件。可以通过下面命令来查看是否创建成功：
 ``` bash
ls -a
```

* 方式2：git clone (从远程仓库克隆一个已经存在的仓库)
使用下面命令可以从远程仓库克隆一个已经存在的仓库：
``` bash
git clone <url>
```
