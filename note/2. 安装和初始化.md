# 初始化：
使用**git config**命令设置用户名和邮箱,用于识别是谁提交的修改，这个命令只需要输入一次：
``` bash
git config --global user.name "your name"
git config --global user.email "your email"
```
> 省略（Local）：本地配置，只对当前仓库有效
> 
> --global：全局配置，对所有仓库有效
> 
> --system：系统配置，对所有用户有效（一般不会使用）

可以使用下面这个命令来保存用户名和邮箱，这样不用每次提交时输入用户名和邮箱：
``` bash
git config --global credential.helper store
```

也可以使用下面的命令来保存用户名和密码：
``` bash
git config --global --list
```

配置完之后就可以使用Git管理内容和源代码了。
